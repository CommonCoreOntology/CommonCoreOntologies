Location of the CCO IRI registry. 


Instructions for developers who are creating new terms as part of a pull request.

1. Working on a branch, open the file located in this directory and navigate to the bottom.
2. Add your name, the date, and associated issue to the next available IRI(s).
3. As part of your PR, make a separate commit for just the chnages to the iri registry. This should be your last commit before making the PR.
